Features

為台灣使用者打造

🔍

自動偵測英文信件

智慧判斷信件內容語言，英文信件開啟時自動顯示翻譯按鈕，中文信件則不會打擾。

🇹🇼

台灣繁體中文

翻譯結果採用台灣慣用語和繁體中文，而非簡體或港式用語，讀起來更自然。

⚡

就地翻譯，不跳轉

翻譯結果直接取代原文顯示，不需要開新分頁或複製文字到翻譯網站。

🔄

一鍵切換原文

翻譯後可隨時點擊「顯示原文」切回英文，方便對照確認。

📋

長信件批次翻譯

自動將長信件分段處理，即使是超長的電子報或討論串也能完整翻譯。

🔨

可拖拉浮動按鈕

翻譯按鈕可以自由拖拉到畫面中任何位置，不會擋住重要內容。

Installation Guide

安裝方法指引

1

下載擴充功能

點擊下方按鈕下載 gmail-translator-v1.4.0.zip 壓縮檔，或從 GitHub 取得最新版本。

⬇ 下載 .zip GitHub Repo
2

解壓縮檔案

將下載的 .zip 檔案解壓縮到任意資料夾。解壓縮後應包含以下檔案：

gmail-translator-v1.4.0/
  manifest.json
  background.js
  content.js
  popup.html
  style.css
  icon48.png / icon128.png
💡 你不需要修改以上任何檔案內容，確認解壓縮完成後可直接前往 Step 3。
3

開啟 Chrome 擴充功能頁面

在 Chrome 瀏覽器網址列輸入以下網址，進入擴充功能管理頁面：

chrome://extensions ↗ 前往

開啟右上角的 開發人員模式 (Developer mode) 開關。

4

載入擴充功能

點擊左上角 載入未封裝項目 (Load unpacked) 按鈕，選擇剛才解壓縮的資料夾。

載入成功後，擴充功能列表中會出現「Gmail 一鍵翻譯 — 台灣慣用語中文」。

5

開始使用

前往 Gmail 打開任何一封英文信件。如果偵測到英文內容，右上角會自動浮現藍色的 🌐 翻譯成中文 按鈕，點擊即可翻譯。

翻譯完成後可點擊 🔄 顯示原文 切回英文。

⚠️
注意：此擴充功能使用 Chrome Manifest V3 架構，僅支援 Google Chrome 及基於 Chromium 的瀏覽器（Edge、Brave、Arc 等）。每次 Chrome 更新後，開發者模式載入的擴充功能可能需要重新啟用。
